What is SB Web Kids?

We are Santa Barbara's premier web design camp for kids age 13-19. SB Web Kids leads small groups through a hybrid curriculum that mixes live coding exercises with popular online courses (coached in-person). Students learn advanced online problem solving and build real customized websites using HTML and CSS. No prior experience is necessary.
